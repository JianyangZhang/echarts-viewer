json.textStyle | object | 全局字体样式

textStyle.fontFamily | string | 字体

textStyle.fontSize | int | 字体大小

textStyle.color | string | 字体颜色

textStyle.fontStyle | "normal" "italic" "oblique" | 字体风格

textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 字体粗细

json.backgroundColor | string | 容器背景颜色

json.backgroundColor | string | 容器背景颜色，默认为"transparent"透明

json.title | object | 主标题

title.show | boolean | 是否显示标题组件

title.text | string | 标题文本，支持\n换行

title.textStyle | object | 文本样式设置

  textStyle.fontFamily | string | 主题文本字体

  textStyle.fontSize | int | 主题文本大小

  textStyle.color | string | 主题文本颜色

  textStyle.fontStyle | "normal" "italic" "oblique" | 主题文本风格

  textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 主题文本粗细

title.subtext | string | 副标题文本，支持\n换行

title.padding | [int, int, int, int] | 标题内边距

title.left | int 或 "int%" 或 "left" "center" "right"| 标题距离容器左侧的距离

title.right | int 或 "int%" | 标题距离容器右侧的距离

title.top | int 或 "int%" 或 "top" "middle" "bottom" | 标题距离容器顶端的距离

title.bottom | int 或 "int%" | 标题距离容器底端的距离

title.backgroundColor | string | 标题背景色

title.borderColor | string | 标题边框颜色

title.borderWidth | int | 标题边框粗细

title.zlevel | int | zlevel值，优先级高于z值，每个zlevel值对应一个Canvas层

title.z | int | z值，优先级低于zlevel值，在同一个Canvas层中再分层

json.toolbox | object | 工具栏

toolbox.show | boolean | 是否显示工具栏组件

toolbox.orient | "horizontal" "vertical" | 工具栏的布局朝向

toolbox.itemSize | int | 工具栏图标大小

toolbox.itemGap | int | 工具栏图标间隔

toolbox.showTitle | boolean | 是否在鼠标hover时显示按钮功能

toolbox.feature | object | 工具栏的功能

  feature.restore | object | 还原配置项为默认值

    restore.show | boolean | 是否显示此工具

  feature.saveAsImage | object | 保存此图表为图片

    saveAsImage.show | boolean | 是否显示此工具

    saveAsImage.type | "png" "jpeg" | 保存的图片格式

    saveAsImage.name | string | 保存的图片文件名

    saveAsImage.backgroundColor | string | 保存的图片背景颜色

    saveAsImage.pixelRatio | int | 保存的图片分辨率比例，默认为1，即跟容器的分辨率相同

  feature.dataView | object | 数据视图工具，可以展现当前图表所用的数据，编辑后可以动态更新。

    dataView.show | boolean | 是否显示此工具

    dataView.readOnly | boolean | 是否只读

    dataView.backgroundColor | string | 数据视图边栏颜色

    dataView.textareaColor | string | 数据视图文本输入区背景颜色

    dataView.textareaBorderColor | string | 数据视图文本输入区边框颜色

    dataView.textColor | string | 数据视图文本颜色

    dataView.buttonColor | string | 数据视图按钮颜色

    dataView.buttonTextColor | string | 数据视图按钮文本颜色

  feature.dataZoom | object | 区域缩放，目前只支持坐标型图表的缩放

    dataZoom.show | boolean | 是否显示此工具

    dataZoom.xAxisIndex | boolean 或 int 或 array | 例如: false:不控制x轴 3:只控制axisIndex为3的x轴 [0, 3]:只控制axisIndex为0和3的x轴

    dataZoom.yAxisIndex | boolean 或 int 或 array | 例如: false:不控制y轴 3:只控制axisIndex为3的y轴 [0, 3]:只控制axisIndex为0和3的y轴

toolbox.left | int 或 "int%" 或 "left" "center" "right"| 工具栏距离容器左侧的距离

toolbox.right | int 或 "int%" | 工具栏距离容器右侧的距离

toolbox.top | int 或 "int%" 或 "top" "middle" "bottom" | 工具栏距离容器顶端的距离

toolbox.bottom | int 或 "int%" | 工具栏距离容器底端的距离

toolbox.zlevel | int | zlevel值，优先级高于z值，每个zlevel值对应一个Canvas层

toolbox.z | int | z值，优先级低于zlevel值，在同一个Canvas层中再分层

toolbox.width | int 或 "auto" | 工具栏的宽度，默认auto自适应

toolbox.height | int 或 "auto" | 工具栏的高度，默认auto自适应

json.xAxis / json.yAxis | object | 坐标轴相关配置

xAxis.position | "top" "bottom" | x轴的位置 / yAxis.position | "left" "right" | y轴的位置

xAxis.offset / yAxis.offset | int | 坐标相对于默认位置的偏移，相同的position有多个坐标轴的时候有用

xAxis.type / yAxis.type | "value" "category" "time" "log" | 坐标轴的类型，value适用于连续数据，category适用于离散数据，time适用于连续的时序数据，log适用于对数数据

xAxis.name / yAxis.name | string | 坐标轴名称

xAxis.nameLocation / yAxis.nameLocation | "start" "middle" "end" | 坐标轴名称显示位置

xAxis.nameTextStyle / yAxis.nameTextStyle | object | 坐标轴名称的文字样式

  nameTextStyle.fontFamily | string | 坐标轴名称文本字体

  nameTextStyle.fontSize | int | 坐标轴名称文本大小

  nameTextStyle.color | string | 坐标轴名称文本颜色

  nameTextStyle.fontStyle | "normal" "italic" "oblique" | 坐标轴名称文本风格

  nameTextStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 坐标轴名称文本粗细

xAxis.nameGap / yAxis.nameGap | int | 坐标轴名称与轴线之间的距离

xAxis.inverse / yAxis.inverse | boolean | 是否是反向坐标轴

xAxis.min / yAxis.min | int 或 "auto" "dataMin" | 坐标轴刻度最小值，auto会自动计算最小值保证坐标轴刻度的均匀分布，dataMin会取数据在该轴上的最小值作为最小刻度

xAxis.max / yAxis.max | int 或 "auto" "dataMax" | 坐标轴刻度最大值，auto会自动计算最大值保证坐标轴刻度的均匀分布，dataMax会取数据在该轴上的最大值作为最大刻度

xAxis.scale / yAxis.scale | boolean | 只对type为value的图表有效，设置成true后，坐标刻度不会强制包含0刻度

xAxis.minInterval / yAxis.minInterval | int | 只对type为value的图表有效，坐标轴的最小间隔大小，比如可以设置成1来保证坐标轴分割刻度显示成整数

xAxis.splitNumber / yAxis.splitNumber | int | 在type为category的图表中无效，坐标轴的分割段数，只是个预估值，最后实际显示的段数可能会自动做调整

xAxis.interval / yAxis.interval | int | 设置坐标轴分割间隔，因为splitNumber可能会自动做调整，可以使用interval配合min max强制设定刻度划分（不建议使用）
