"inside"是内置型数据区域缩放工具，它的常用配置项有：

  dataZoom[index].disabled | boolean | 是否禁用此功能

  dataZoom[index].xAxisIndex | int 或 array | 设置此组件控制的x轴，默认控制和dataZoom平行的第一个xAxis。例如：

  xAxis: [{第一个x轴}, {第二个x轴}, {第三个x轴}], dataZoom: [{xAxisIndex[0, 2]}, {xAxisIndex: 1}]

  这表示有三个x轴，两个dataZoom组件。第一个dataZoom组件控制第一个和第三个x轴，第二个dataZoom组件控制第二个x轴

  dataZoom[index].disabled | boolean | 是否禁用此功能

  dataZoom[index].yAxisIndex | int 或 array | 设置此组件控制的y轴，默认控制和dataZoom平行的第一个yAxis。例如：

  yAxis: [{第一个y轴}, {第二个y轴}, {第三个y轴}], dataZoom: [{yAxisIndex[0, 2]}, {yAxisIndex: 1}]

  这表示有三个y轴，两个dataZoom组件。第一个dataZoom组件控制第一个和第三个y轴，第二个dataZoom组件控制第二个y轴

  dataZoom[index].filterMode | "filter" "empty" |

  "filter":当前数据窗口外的数据被过滤掉，即会影响其他轴的数据范围；"empty":当前数据窗口外的数据被设置为空，即不会影响其他轴的数据范围

  dataZoom[index].start | int | 数据窗口范围的起始百分比，范围是：0 ~ 100，表示 0% ~ 100%

  dataZoom[index].end | int | 数据窗口范围的结束百分比，范围是：0 ~ 100，表示 0% ~ 100%

  dataZoom[index].startValue | int 或 String 或 Date | 数据窗口范围的起始值，如果设置了.start则.startValue失效

  对于category轴，startValue既可以设置为axis.data数组的index，也可以设置为数组值本身

  dataZoom[index].endValue | int 或 String 或 Date | 数据窗口范围的结束值，如果设置了.end 则.endValue失效
