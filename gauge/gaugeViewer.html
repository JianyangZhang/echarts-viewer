主标题 (json.title)说明

json.title

json.title.show | boolean | 是否显示标题组件

json.title.text | string | 标题文本，支持\n换行

json.title.textStyle | object

  textStyle.color | string | 主题文本颜色

  textStyle.fontStyle | "normal" "italic" "oblique" | 主题文本风格

  textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400...| 主题文本粗细

  textStyle.fontFamily | string | 主题文本字体

  textStyle.fontSize | int | 主题文本大小

json.title.subtext | string | 副标题文本，支持\n换行

json.title.padding | [int, int, int, int] | 标题内边距

json.title.left | int 或 "int%" 或 "left" "center" "right"| 标题距离容器左侧的距离

json.title.right | int 或 "int%" | 标题距离容器右侧的距离

json.title.top | int 或 "int%" 或 "top" "middle" "bottom" | 标题距离容器顶端的距离

json.title.bottom | int 或 "int%" | 标题距离容器底端的距离

json.title.backgroundColor | string | 标题背景色

json.title.borderColor | string | 标题边框颜色

json.title.borderWidth | int | 标题边框粗细

json.title.zlevel | int | zlevel值，优先级高于z值，每个zlevel值对应一个Canvas层

json.title.z | int | z值，优先级低于zlevel值，在同一个Canvas层中再分层
