json.textStyle | object | 全局字体样式

textStyle.fontFamily | string | 字体

textStyle.fontSize | int | 字体大小

textStyle.color | string | 字体颜色

textStyle.fontStyle | "normal" "italic" "oblique" | 字体风格

textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 字体粗细

json.backgroundColor | string | 容器背景颜色

backgroundColor | string | 容器背景颜色，默认为"transparent"透明

json.title | object | 主标题

title.show | boolean | 是否显示标题组件

title.text | string | 标题文本，支持\n换行

title.textStyle | object | 文本样式设置

  textStyle.fontFamily | string | 主题文本字体

  textStyle.fontSize | int | 主题文本大小

  textStyle.color | string | 主题文本颜色

  textStyle.fontStyle | "normal" "italic" "oblique" | 主题文本风格

  textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 主题文本粗细

title.subtext | string | 副标题文本，支持\n换行

title.padding | [int, int, int, int] | 标题内边距

title.left | int 或 "int%" 或 "left" "center" "right"| 标题距离容器左侧的距离

title.right | int 或 "int%" | 标题距离容器右侧的距离

title.top | int 或 "int%" 或 "top" "middle" "bottom" | 标题距离容器顶端的距离

title.bottom | int 或 "int%" | 标题距离容器底端的距离

title.backgroundColor | string | 标题背景色

title.borderColor | string | 标题边框颜色

title.borderWidth | int | 标题边框粗细

title.zlevel | int | zlevel值，优先级高于z值，每个zlevel值对应一个Canvas层

title.z | int | z值，优先级低于zlevel值，在同一个Canvas层中再分层

json.toolbox | object | 工具栏

toolbox.show | boolean | 是否显示工具栏组件

toolbox.orient | "horizontal" "vertical" | 工具栏的布局朝向

toolbox.itemSize | int | 工具栏图标大小

toolbox.itemGap | int | 工具栏图标间隔

toolbox.showTitle | boolean | 是否在鼠标hover时显示按钮功能

toolbox.feature | object | 工具栏的功能

  feature.restore | object | 还原配置项为默认值

    restore.show | boolean | 是否显示此工具

  feature.saveAsImage | object | 保存此图表为图片

    saveAsImage.show | boolean | 是否显示此工具

    saveAsImage.type | "png" "jpeg" | 保存的图片格式

    saveAsImage.name | string | 保存的图片文件名

    saveAsImage.backgroundColor | string | 保存的图片背景颜色

    saveAsImage.pixelRatio | int | 保存的图片分辨率比例，默认为1，即跟容器的分辨率相同

  feature.dataView | object | 数据视图工具，可以展现当前图表所用的数据，编辑后可以动态更新。

    dataView.show | boolean | 是否显示此工具

    dataView.readOnly | boolean | 是否只读

    dataView.backgroundColor | string | 数据视图边栏颜色

    dataView.textareaColor | string | 数据视图文本输入区背景颜色

    dataView.textareaBorderColor | string | 数据视图文本输入区边框颜色

    dataView.textColor | string | 数据视图文本颜色

    dataView.buttonColor | string | 数据视图按钮颜色

    dataView.buttonTextColor | string | 数据视图按钮文本颜色

  feature.dataZoom | object | 区域缩放，目前只支持坐标型图表的缩放

    dataZoom.show | boolean | 是否显示此工具

    dataZoom.xAxisIndex | boolean 或 int 或 array | 例如: false:不控制x轴；3:只控制axisIndex为3的x轴；[0, 3]:只控制axisIndex为0和3的x轴

    dataZoom.yAxisIndex | boolean 或 int 或 array | 例如: false:不控制y轴；3:只控制axisIndex为3的y轴；[0, 3]:只控制axisIndex为0和3的y轴

toolbox.left | int 或 "int%" 或 "left" "center" "right"| 工具栏距离容器左侧的距离

toolbox.right | int 或 "int%" | 工具栏距离容器右侧的距离

toolbox.top | int 或 "int%" 或 "top" "middle" "bottom" | 工具栏距离容器顶端的距离

toolbox.bottom | int 或 "int%" | 工具栏距离容器底端的距离

toolbox.zlevel | int | zlevel值，优先级高于z值，每个zlevel值对应一个Canvas层

toolbox.z | int | z值，优先级低于zlevel值，在同一个Canvas层中再分层

toolbox.width | int 或 "auto" | 工具栏的宽度，默认auto自适应

toolbox.height | int 或 "auto" | 工具栏的高度，默认auto自适应

json.series[index].type | "gauge" | series数组的每个type为gauge的元素是一个仪表盘主体

series[index].name | string | 仪表盘名称

series[index].radius | int 或 "int%" | 仪表盘半径

series[index].startAngle | int | 仪表盘起始角度，默认单位为 °

series[index].endAngle | int | 仪表盘结束角度，默认单位为 °

series[index].clockwise | boolean | 仪表盘刻度是否为顺时针增长

series[index].min | int | 最小刻度值

series[index].max | int | 最大刻度值

series[index].splitNumber | int | 默认为10，值越大刻度线分割得越细

series[index].axisLine | object | 轴线相关配置

  axisLine.show | boolean | 是否显示轴线(测试无效)

  axisLine.lineStyle | object | 轴线样式

    lineStyle.color | array | 每段的结束位置和颜色可以通过一个数组来表示，例如: [[0.2, '#91c7ae'], [0.8, '#63869e'], [1, '#c23531']]

    lineStyle.width | int | 轴线宽度

    lineStyle.opacity | int | 透明度，范围是0到1，0代表完全透明，1代表完全不透明

series[index].splitLine | object | 分隔线相关配置（分隔线是把整体轴线等分为几大份）

  splitLine.show | boolean | 是否显示分隔线

  splitLine.length | int 或 "int%" | 分隔线线长

  splitLine.lineStyle | object | 分隔线样式

    lineStyle.color | string | 分隔线颜色

    lineStyle.width | int | 分隔线宽度

    lineStyle.type | "solid" "dashed" "dotted" | 分隔线类型

    lineStyle.opacity | int | 透明度，范围是0到1，0代表完全透明，1代表完全不透明

series[index].axisTick | object | 刻度线相关配置

  axisTick.show | boolean | 是否显示刻度线

  axisTick.splitNumber | int | 默认是5，值越大刻度线分割得越细

  axisTick.length | int 或 "int%" | 刻度线长

  splitLine.lineStyle | object | 刻度线样式

    lineStyle.color | string | 刻度线颜色

    lineStyle.width | int | 刻度线宽度

    lineStyle.type | "solid" "dashed" "dotted" | 刻度线类型

    lineStyle.opacity | int | 透明度，范围是0到1，0代表完全透明，1代表完全不透明

series[index].axisLabel | object | 刻度标签相关配置，刻度标签即刻度线旁边的数字

  axisLabel.show | boolean | 是否显示刻度标签

  axisLabel.distance | int | 刻度标签与刻度线的距离

  axisLabel.textStyle | object | 刻度标签文本样式

    textStyle.fontFamily | string | 刻度标签字体

    textStyle.fontSize | int | 刻度标签字体大小

    textStyle.color | string | 刻度标签文本颜色

    textStyle.fontStyle | "normal" "italic" "oblique" | 刻度标签字体风格

    textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 刻度标签字体粗细

series[index].pointer | object | 仪表盘指针

  pointer.show | boolean | 是否显示指针

  pointer.length | int 或 "int%" | 指针长度

  pointer.width | int | 指针宽度

series[index].title | object | 仪表盘内部标题

  title.show | boolean | 是否显示仪表盘内部标题

  title.offsetCenter | array | 仪表盘内部标题相对于仪表盘中心的偏移位置，可以是int或"int%"，例如[0, '-40%']: 第一项是水平方向的偏移，第二项是垂直方向的偏移

  title.textStyle | object | 仪表盘内部标题文本样式

    textStyle.fontFamily | string | 仪表盘内部标题字体

    textStyle.fontSize | int | 仪表盘内部标题字体大小

    textStyle.color | string | 仪表盘内部标题文本颜色

    textStyle.fontStyle | "normal" "italic" "oblique" | 仪表盘内部标题字体风格

    textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 仪表盘内部标题字体粗细

series[index].details | object | 仪表盘内部数字，显示此时仪表盘的数值

  details.show | boolean | 是否显示仪表盘内部数字

  details.width | int | 仪表盘内部数字的容器宽度

  details.height | int | 仪表盘内部数字的容器高度

  details.backgroundColor | string | 仪表盘内部数字的容器背景颜色

  details.borderWidth | int | 仪表盘内部数字的容器边框宽度

  details.borderColor | string | 仪表盘内部数字的容器边框颜色

  details.offsetCenter | array | 仪表盘内部数字相对于仪表盘中心的偏移位置，可以是int或"int%"，例如[0, '-40%']: 第一项是水平方向的偏移，第二项是垂直方向的偏移

  details.textStyle | object | 仪表盘内部数字文本样式

    textStyle.fontFamily | string | 仪表盘内部数字字体

    textStyle.fontSize | int | 仪表盘内部数字字体大小

    textStyle.color | string | 仪表盘内部数字文本颜色

    textStyle.fontStyle | "normal" "italic" "oblique" | 仪表盘内部数字字体风格

    textStyle.fontWeight | "normal" "bold" "bolder" "lighter" 或 100 200 300 400... | 仪表盘内部数字字体粗细
